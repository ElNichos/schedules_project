About project

This site is designed to generate and track class schedules for educational institutions. The project is developed for the sole purpose of learning the Django framework and may not be exploitable. This is a beta version of a non-commercial project, so the author does not bear any responsibility for possible technical failures.

It is assumed that a non-authorized user is a student or teacher. Such a user can only view the schedule.

To view the schedule, go to the "Teacher's" or "Students's" tab on the navigation bar or use the cards on the home page. Then it is necessary to fill in the form by entering the name of the group/teacher and your institution. You should also specify: semester or session schedule. After submitting the form you will be redirected to the schedule page.

To switch the week of the schedule, click on the edge of the page.

When viewing the schedule, the current lesson will be highlighted in blue color.

If the user is authorized, in addition to viewing schedules, he/she can also create teacher, group and lesson objects.

On the sidebar you can access "Action panel". Here you can create students and teachers objects, check if there are objects in the database, display the table of existing objects, change the number of week from which the time will be counted down.

When viewing the page with timetable you can create lessons with specified lesson number, day and week number.
